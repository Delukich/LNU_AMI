# LNU_AMI

Цей репозиторій містить **лабораторні роботи та практичні завдання**, виконані під час навчання у
**Львівському національному університеті імені Івана Франка** на **факультеті прикладної математики та інформатики**.

Освітня програма — **Комп’ютерні науки**.

Матеріали згруповані за курсами навчання (**1 курс**, **2 курс**, **3 курс**, **4 курс**). У кожній папці зібрані роботи з відповідних дисциплін, що охоплюють основи програмування, алгоритми, структури даних та інші предмети, передбачені навчальним планом.
